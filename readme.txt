Adds some extra functionality to the Shopp admin to make your life easier!

== Description ==

This plugin tries to help with a few admin tasks that can be time consuming otherwise. It allows you to:

* Navigate between pending orders easily
* Change the order status on the order page
* Enable/Disable Comments and Pings on all Shopp Products

Got a feature you'd like to see added to this plugin? Let me know!

== Installation ==

Upload the Shopp Admin Extras plugin to your blog, then activate it. You're done!
